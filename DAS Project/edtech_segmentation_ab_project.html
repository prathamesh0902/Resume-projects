ML Pipeline — From Raw Scores to Segments

Feature Engineering

Pandas + dbt
→
02

scikit-learn
→
03
Clustering

k=5 · sil=0.71
→
04
Segment Scoring
Random Forest classifier trained on labeled cohorts (F1=0.88), LTV regression per segment, churn propensity
RF · XGBoost
→
05
Experiment Design
Power analysis per segment (α=0.05, β=0.20), stratified random assignment by school & grade
Bayesian + freq.
